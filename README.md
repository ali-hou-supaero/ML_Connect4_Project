# ML_Connect4_Project
Machine learning project on a Connect 4 database that can be found here : https://archive.ics.uci.edu/dataset/26/connect+4

## Requirements
⚠️ This repo uses Git LFS to store large model files.  
To clone properly, make sure you have Git LFS installed and running.
### 1 - Install Git LFS
```bash
sudo apt install git-lfs
```
### 2 - - Initialize Git LFS (once per machine):
```bash
git lfs install
```
